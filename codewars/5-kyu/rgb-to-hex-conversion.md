## #1 first answer

```js
const rgb = (r, g, b) => [r, g, b]
 .map((color) => (
    color < 0 ? 0 :
    color > 255 ? 255
    : color)
    .toString(16)
    .padStart(2, "0")
    .toUpperCase()
  ).join("")
```
